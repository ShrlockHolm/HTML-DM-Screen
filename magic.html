STACKING

Different spells with the same effect stack. The same spell cast multiple times takes the highest bonus.

SPELLCASTING IN ARMOR

Requires proficiency in that armor. Wearing armor without proficiency prevents spellcasting.

COPYING A SPELL INTO A SPELLBOOK

A wizard can add a found spell to their spellbook if it is of a level for which they have spell slots and if they can spare the time.

SPELL SLOTS, KNOWN SPELLS, AND PREPARED SPELLS

Each spellcasting class has a limit to the number of spells they can cast per day. The number varies by level and class. Tables detailing the number of available spell slots can be found in the class description. Typically a spellcaster can prepare a number of spells equal to their spellcasting ability plus thier proficiency bonus
